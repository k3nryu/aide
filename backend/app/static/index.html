财务概况

现金流法

公式：每月被动收入 >= 每月固定支出。用于判断你现在是否已经不用主动工作也能覆盖生活开销。
0% 被动收入 / 固定支出 看不用工作的收入能覆盖多少生活成本。
0 固定支出项目合计 由房租、饮食、水电网等每月支出项目加总。
0 无需主动工作的收入 例如股息、房租、利息等被动现金流。
0 被动收入 - 固定支出 为负数表示每月还需要工资或其他主动收入补足。

本金法

公式：FI目标本金 = 年固定支出 ÷ 安全提取率。用于估算需要准备多少可投资本金。
0 FI目标本金 按“年固定支出 ÷ 安全提取率”算出来。
0 FI本金 能产生现金流或未来投资收益的资产。
0 FI目标本金 - FI资产 还差多少本金。
0 每月新增投资 用来估算把本金缺口补齐还需要多久。
